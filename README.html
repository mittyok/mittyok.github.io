NexT.Pisces

Hexo主题NexT.Pisces的Typecho移植版，基于zgq354的NexT.Mist修改制作

在线预览

使用方法

  1. 点击download zip下载最新源码，修改文件名为NexTPisces，解压上传
  2. 后台启用
  3. 新建分类页，缩略名为categories，自定义模板选择categories
  4. 新建归档页，缩略名为archive，自定义模板选择archives
  5. 新建标签页，缩略名为tags，自定义模板选择tags
  6. 后台配置外观选项，包括头像、昵称、站点描述、侧边栏相关设置

注意事项

  • 侧边栏友链只有在安装了 Hanny 的 Links插件 才可显示，未安装不影响使用
